# AI-Powered Service Booking Platform

A full-stack web application built with the MERN stack (MongoDB, Express, React, Node.js) and Tailwind CSS. Connects users with service providers using an AI-fueled smart suggestion system.

## 🌟 Features
- **AI Service Suggestions:** Just explain your problem ("mera AC thanda nahi kar raha") and the AI will suggest the correct service.
- **Role-Based Access Control:** Distinct dashboards for 'Users', 'Providers', and 'Admins'.
- **Modern UI:** Built with Tailwind CSS, utilizing glassmorphism and premium aesthetics.
- **RESTful API:** Structured backend API using Express and nested routes.

## 🛠️ Technologies Used
- React + Vite
- Tailwind CSS
- Node.js + Express
- MongoDB + Mongoose
- JSON Web Tokens (JWT) for Auth

## 🚀 Setup Steps

### 1. Backend Setup
1. Open a terminal and navigate to `cd backend`.
2. Install NodeJS dependencies: `npm install`
3. Configure the `backend/.env` file with a valid `MONGO_URI`. By default, it connects to local MongoDB on `27017`.
4. Start the server (runs on `http://localhost:5000`):
   ```bash
   node server.js
   ```

### 2. Frontend Setup
1. Open a new terminal and navigate to `cd frontend`.
2. Install dependencies: `npm install`
3. Start the Vite React development server:
   ```bash
   npm run dev
   ```
